# Product Description Genius - Classic Version

Αυτή είναι η κλασική έκδοση της εφαρμογής Product Description Genius, που χρησιμοποιεί μόνο HTML, CSS και JavaScript. Η εφαρμογή επιτρέπει τη δημιουργία περιγραφών προϊόντων με βάση τα χαρακτηριστικά τους.

## Τεχνολογίες

Αυτή η έκδοση χρησιμοποιεί:
- HTML5
- CSS3
- Vanilla JavaScript
- Εικονίδια από Font Awesome (μέσω CDN)

Δεν απαιτείται εγκατάσταση npm ή διαδικασία build.

## Χαρακτηριστικά

- Προβολή και διαχείριση προϊόντων
- Αναζήτηση και φιλτράρισμα προϊόντων
- Προσθήκη και επεξεργασία προϊόντων
- Αυτόματη δημιουργία περιγραφών προϊόντων (προσομοίωση)
- Ρυθμίσεις για το API key και τις προτιμήσεις περιεχομένου

## Δομή Έργου

```
/classic-version
  /css
    style.css         # Στυλιστική μορφοποίηση
  /js
    data.js           # Δεδομένα εφαρμογής
    main.js           # Κύριος κώδικας JavaScript
  /images             # Εικόνες (εάν υπάρχουν)
  index.html          # Κύρια σελίδα
  README.md           # Αυτό το αρχείο
```

## Τρόπος Χρήσης

1. Απλά ανοίξτε το αρχείο `index.html` σε οποιοδήποτε σύγχρονο πρόγραμμα περιήγησης.
2. Δεν απαιτείται διακομιστής web ή οποιαδήποτε άλλη εγκατάσταση.

## Προσομοίωση API

Αυτή η έκδοση προσομοιώνει τις κλήσεις API για δημιουργία περιγραφών προϊόντων. Σε πραγματική εφαρμογή, θα συνδεόταν με το OpenAI API.

## Μετατροπή από React

Αυτή η έκδοση είναι μια μετατροπή από την αρχική εφαρμογή React. Η λειτουργικότητα έχει διατηρηθεί αλλά η υλοποίηση έχει απλοποιηθεί για χρήση με τις βασικές τεχνολογίες web.

## Περαιτέρω Ανάπτυξη

Για να επεκτείνετε αυτή την εφαρμογή:

1. Προσθέστε πραγματική σύνδεση με το OpenAI API
2. Υλοποιήστε τοπική αποθήκευση με το localStorage API
3. Προσθέστε περισσότερες δυνατότητες επεξεργασίας περιγραφών
4. Βελτιώστε την απόδοση για μεγαλύτερες λίστες προϊόντων
